# Spot Exposure Hedging Bot

A sophisticated risk management system that monitors and automatically hedges directional risk of spot positions using perpetual futures or options. Controlled via Telegram bot with interactive interface.

## Features

- Real-time risk monitoring (Delta, Gamma, Theta, Vega, VaR)
- Multiple hedging strategies (delta-neutral, protective puts, covered calls, dynamic)
- Telegram bot interface with interactive buttons and alerts
- Smart order routing across multiple exchanges
- Comprehensive risk analytics and reporting
